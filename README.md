# Calculator

A fully functional calculator application.

## Setup

### Prerequisites

- Node.js 
- npm 

### Installation

1. Clone the repository:
   ```sh
   git clone https://github.com/Kanishka227/Calculator.git
   
2. cd Calculator

3. npm install

4. npm run dev

5. The server will start at http://localhost:3000.
